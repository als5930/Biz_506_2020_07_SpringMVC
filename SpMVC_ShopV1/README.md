# Spring Project 빛나리 쇼핑몰V1
* 상품관리, 거해처관리, 회원가입, 로그인을 포람
* DB 오라클
* 반응형 메뉴를 사용, 반응형 메인화면 구현

## Project 시작
* Java Version 1.8로 변경
* Spring Framework 5.2.8
* lombock, logback
* views/home.jsp 삭제후 재 생성

*spring-jdbc
*mybatis
*mybtis-spring

* ojdbc6
